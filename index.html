Extract valuable business information from any website with our advanced AI-powered tool

Project Overview

This AI-powered web scraper analyzes websites and extracts business information, technologies used, SEO data, and more.

Key Features:

  • Extract business information including contact details, services, and products
  • Identify technologies and frameworks used on the website
  • Analyze SEO data like meta tags, headings, and image alt attributes
  • Examine page structure and navigation elements
  • Download results in JSON, CSV, or text formats

How to Use

To run this project locally:

  1. Clone the repository: git clone https://github.com/gcebece/webscrapper.git
  2. Install dependencies: npm install
  3. Start the development server: npm run dev
  4. Open http://localhost:3000 in your browser
  5. Enter a website URL and click "Analyze Website"
